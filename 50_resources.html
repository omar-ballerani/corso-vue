Vue-resource è un client http che si può utilizzare per inviare richieste http.

Link: https://github.com/pagekit/vue-resource

Dopo aver caricato lo script, il servizio http si può utilizzare o globalmente facendo riferimento a Vue.http o in ogni istanza Vue tramite this.$http

Per i nostri esempi ed esercizi utilizzeremo questi services finti: https://jsonplaceholder.typicode.com/
Tutti i metodi che inviano richieste http ritornano una Promise

La sintassi standard che dovete utilizzare è this.$http.get('/someUrl').then(successCallback,errorCallback)

Potete configurare parametri standard in questo modo: Configuration

Per effettuare request http questa è la guida di riferimento: Http request / response
