Getting started

Vue.js è un framework per costruire interfacce utente in maniera dinamica. A differenza di altri framework monolitici, è un framework costituito da diverse librerie che si possono utilizzare o meno.

La libreria core (vue.js) si preoccupa solo della view, ovvero della user interface, e può essere utilizzata da sola per fare data-binding dei propri oggetti javascript con il DOM, senza dunque utilizzare librerie come JQuery per aggiornare il DOM quando i dati cambiano e viceversa.

Oltre alla parte di view, aggiungendo altre librerie facenti parte sempre del framework Vue (router, loader, etc) è possibile arrivare a costruire Single Page Application molto efficienti.
