MVVM

Il Model View ViewModel è un design pattern utilizzato per costruire interfacce utente. Descrive come mantenere semplice una sofisticata UI dividendola in tre componenti:

  • Model: il vostro persistito Domain Model, tipicamente in un'applicazione Javascript richiamerete dei servizi per leggere questi dati e per aggiornarli

  • ViewModel: una rappresentazione in codice dei dati e delle operazioni presenti nella vostra UI; per esempio se state implementando una lista tipicamente avrete un Array contenente i vari elementi e dei metodi per aggiungere / aggiornare / eliminare questi elementi.

  • View: un'interfaccia interattiva e visible che rappresenta lo stato del vostro View Model. Mostra dati del View Model ed invia comandi a questo, si aggiorna quando lo stato del View Model cambia
